================================================================================
                        CONTACT FORM - QUICK START
================================================================================

⚠️ REQUIRED FIRST STEP: Update backend/.env with your credentials

================================================================================
QUICK SETUP (5 MINUTES)
================================================================================

STEP 1: Edit backend/.env
   File: c:\JeffCV\backend\.env
   
   Update these values:
   - SUPABASE_URL: Your Supabase project URL
   - SUPABASE_KEY: Your Supabase anon key
   - EMAIL_USER: your-email@gmail.com
   - EMAIL_PASS: Your Gmail app-specific password (not your regular password!)
   - EMAIL_FROM: your-email@gmail.com
   - EMAIL_TO: Email address to receive contact messages
   - CORS_ORIGIN: http://localhost:3000 (for local dev)

STEP 2: Start Backend
   Terminal 1:
   cd c:\JeffCV\backend
   npm install
   npm run dev
   
   You should see: ✅ Supabase PostgreSQL connected

STEP 3: Start Frontend
   Terminal 2:
   cd c:\JeffCV\frontend
   npm run dev
   
   You should see: ▲ Ready in XXXms

STEP 4: Test
   1. Open http://localhost:3000/contact
   2. Fill in the form with valid data
   3. Click "Send Message"
   4. Open browser DevTools (F12)
   5. Check Console tab - should show:
      ✅ Contact response: {success: true, ...}

================================================================================
EMAIL VALIDATION
================================================================================

Valid Email Examples:
  ✓ user@example.com
  ✓ john.doe@company.co.uk
  ✓ test+tag@domain.com

Invalid Email Examples (will show error):
  ✗ user@
  ✗ user
  ✗ @example.com
  ✗ user@example (missing extension)

The form shows validation feedback in real-time:
  - While typing: "✓ Valid email format" (green) OR "✗ Invalid email" (red)
  - Submit button only works if email is valid

================================================================================
IF IT DOESN'T WORK
================================================================================

1. Check Backend Console Output
   - Look for 📧 Processing contact form submission
   - Look for ✅ Message saved to database
   - Look for ❌ errors

2. Check Browser Console (F12 → Console)
   - Should see 📧 Sending contact form
   - Should see ✅ Contact response

3. Check Your Credentials
   - Are SUPABASE_URL and SUPABASE_KEY correct?
   - Is EMAIL_USER and EMAIL_PASS correct?
   - Gmail users: Did you use app-specific password?

4. Check CORS
   - Is frontend running on http://localhost:3000?
   - Does backend .env have CORS_ORIGIN=http://localhost:3000?

5. Check Database
   - Does contact_messages table exist in Supabase?
   - Can you access your Supabase dashboard?
